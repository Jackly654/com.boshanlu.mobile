## 博山庐手机客户端
博山庐手机客户端是基于西安电子科技大学校园睿思论坛Android客户端开源项目在Apache License V2.0下二次开发的项目，原项目地址：https://github.com/freedom10086/Ruisi
支持夜间模式，支持主题，如有bug，或者新功能建议欢迎提出Issue .
